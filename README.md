# Nest + React Template

This isn't just another starter template; it's a **complete blueprint** I'm building for scalable full-stack apps using **NestJS** on the backend and **React** on the frontend. I'm designing it to be super modular, AI-friendly, and ready for real production work.

## 🚀 The Vision
I want to create a solid foundation with pre-built modules that are easy to manage. Later on, I’m planning to build a **dedicated CLI** to help manage these modules and scaffold new features in seconds.

## 📦 Planned Modules (A few examples)
I'll be adding a library of enterprise-grade modules. Some of the things I'm planning:
- **Full Auth**: A complete auth system with **RBAC (Role-Based Access Control)**.
- **Emailing**: Easy integration for transactional emails.
- **Stripe**: Ready-to-go payment processing and subscriptions.
*And plenty more to come...*

## 🤖 AI-Optimized Development
To make sure AI assistants (like ChatGPT or Claude) actually write good code instead of "slop," I'm including:
- **`skills/` folder**: Specialized scripts and instructions for LLMs.
- **`agents/` folder**: Context files (`Agents.md`) with clear architectural rules so AI development stays smooth and consistent.

## 🛠️ Current Tech Stack
Right now, I'm starting with these de-facto standards:
- **Backend/Frontend**: NestJS & React
- **Styling**: Tailwind CSS
- **Forms & Validation**: React-Hook-Form + Zod
- **State Management**: Redux (RTK Query)

## 🔮 What's Next?
In the future, I'll make it easy to swap these for things like **Zustand** or **TanStack Query**. I also plan to roll out similar templates for the **MERN** stack, **Next.js**, and **TanStack Start**.
